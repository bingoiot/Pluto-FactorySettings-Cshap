# Pluto-FactorySettings-Cshap

* Pluto-FactorySettings-Cshap用于给吉风物联网设备烧写服务器SN 
  
	如果你想要获取Pluto-FactorySettings-Cshap的源码:  
	git clone https://github.com/bingoiot/Pluto-FactorySettings-Cshap.git

* 目前支持ESP32、ESP8266芯片烧写，通过芯片自带的Debug信息输出串口烧写,烧写过程 
  不会影响原调试信息的输出，使用此功能无需关闭调试信息输出，直接使用即可。
